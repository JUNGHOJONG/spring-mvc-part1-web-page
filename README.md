# 스프링 mvc 웹 페이지 만들기

## 요구사항 분석
**상품 도메인 모델**
* 상품 ID
* 상품명
* 가격
* 수량

**상품 관리 기능**
* 상품 목록
* 상품 상세
* 상품 등록
* 상품 수정

## 요구사항 추가

**판매 여부**
* 판매 오픈 여부
* 체크 박스로 선택할 수 있다

**등록 지역**
* 서울, 부산, 제주
* 체크 박스로 다중 선택할 수 있다

**상품 종류**
* 도서, 식품, 기타
* 라디오 버튼으로 하나만 선택할 수 있다

**배송 방식**
* 빠른 배송
* 일반 배송
* 느린 배송
* 셀렉트 박스로 하나만 선택할 수 있다
